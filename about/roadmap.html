Project Roadmap 🗺️

Tracking physical office spaces, offline tooling, and pedagogical features shaping the future of COSYlanguages.

🔥 High Priority (The Slow-Tech & Office Era)

Feature / Goal Area Notes
Physical Project Office 🏡 In-Person Space Open a brick-and-mortar COSY languages workspace: an amiable, welcoming community space for quiet studying, a monolingual library, and screen-free face-to-face meetups.
COSYlanguages Book Series 📖 Physical Print Developing comprehensive manuals for English, French, Italian, Russian, and Greek: Grammar Manual, Exercises for Grammar, Vocabulary Manual, and Spoken for Vocabulary workbooks.
COSYlanguages Board Game 🏆 Tabletop Games A custom, premium physical tabletop board game designed to package conversational drills, speaking prompts, and letter-card chains in a social game box.
Printable Leitner Flashcards "Paper Web" Studio A generator allowing users to print 3x5 double-sided vocabulary flashcards straight from their Personal Notebook for offline Leitner box study.
Print-and-Play Board Games Games & Tabletop Printable card decks, game mats, and rulebooks to adapt our digital gaming suite (e.g., Battle of Wits, Action Hero) into tabletop card games.
Audio pronunciation integration Practice & Games Browser-synthesized offline Web Audio native speech support for target vocabulary.

⚡ Medium Priority (Offline & Desktop Tools)

Feature / Goal Area Notes
Mini-Zine folded brochure prints "Paper Web" Studio Turn any 10-unit vocabulary grid or session topic into a printable, single-sheet A4 page that folds into an 8-page pocket glossary.
Local Multi-User Profiles PWA Storage Support independent local dictionary logs, point tallies, and mistake reviews for families sharing the same offline tablet or PC.
Dark Mode theme support Global Layout Toggle a comfortable dark background (preserving warm sand/sage undertones), saved entirely offline in LocalStorage.

✅ Completed & Shipped (Recent Upgrades)

Feature / Goal Area Shipped Details
Student progress tracking Personal Portal Track completed lessons, scores, streaks per language and level via Local Personal Notebook.
Vocabulary Notebook 📓 Personal Portal Implemented a comprehensive personal notebook with custom vocabulary savers, export tools, and review stats.
Monolingual Symbolic Practice 🧩 Practice Hub Transitioned practice structures into fully monolingual layouts utilizing symbolic operators (=, ≈, ≠, 🧩, 🔊) with zero English leakage.
Dynamic Print-CSS Stylesheets Linguistics Completed print-friendly styling rules across morphology tables to let users print beautiful reference guides on physical paper.
Interactive Gamification Audio Language Games Wired native Web Audio Synth sound chimes, buzzes, fanfares, and combo multipliers.
Progressive Web App Offline Caching PWA Foundation Implemented a Service Worker caching foundation (`sw.js`) to load static application codes, icons, stylesheets, and core components without network.
