# images/

Organised by purpose:
- **flags/** — one flag image per language (named by ISO code, e.g. `en.svg`, `el.svg`)
- **logos/** — COSY logo variants
- **games/** — game-specific artwork
- **ui/** — icons and decorative elements
- **courses/** — imagery representing each course type
- **languages/** — hero or illustrative images per language

When adding a new image, place it in the correct subfolder.
Do NOT put images in the root `images/` folder directly.
