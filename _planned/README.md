# _planned — Future features and missing elements

This folder is the single source of truth for everything designed, discussed, or intended
but not yet built on COSYlanguages.

Use this instead of leaving TODO comments scattered across files.
When something is built, remove it from the relevant file here.

Files in this folder:
- planned_features.md — site-wide UI and functional features
- planned_grammar_tables.md — grammar table formats and content gaps
- planned_security.md — security and privacy improvements
- planned_seo.md — SEO and accessibility improvements
- planned_onboarding.md — student/teacher onboarding flows
- planned_pwa.md — PWA and offline improvements
- planned_images.md — images folder audit and reorganisation plan
- planned_languages.md — languages in progress and launch checklist
- planned_courses.md — course types not yet built
- planned_grammar.md — grammar content gaps per language
- planned_vocabulary.md — vocabulary content gaps per language
