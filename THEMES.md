# Approved Theme Taxonomy

The following taxonomy defines the canonical themes and sub-themes for vocabulary data.

| Theme slug      | Label           | Sub-themes (slugs)                              |
|-----------------|-----------------|--------------------------------------------------|
| animals         | Animals         | pets, farm, wild, insects, birds, sea            |
| body            | Body            | face, limbs, organs, health, appearance          |
| clothes         | Clothes         | everyday, formal, accessories, weather_gear      |
| colours         | Colours         | basic, shades                                    |
| emotions        | Emotions        | positive, negative, complex                      |
| food_drink      | Food & drink    | fruit, vegetables, drinks, cooking, meals, snacks|
| furniture       | Furniture       | rooms, appliances, household_items               |
| jobs            | Jobs            | professions, workplace, career                   |
| nature          | Nature          | landscape, weather, plants, seasons              |
| numbers         | Numbers         | cardinal, ordinal, fractions                     |
| people          | People          | family, relationships, nationalities, physical_desc |
| places          | Places          | city, buildings, transport, geography            |
| school          | School          | stationery, subjects, classroom                  |
| shopping        | Shopping        | money, stores, items                             |
| sports          | Sports          | team_sports, individual, gym, outdoor            |
| technology      | Technology      | devices, internet, software                      |
| time            | Time            | clock, days, months, seasons, periods            |
| travel          | Travel          | transport, accommodation, tourism                |
| work            | Work            | office, meetings, documents, business_lang       |
| music           | Music           | instruments, genres, performance                 |
| art_culture     | Art & culture   | art, literature, film, theatre                   |
| health_medicine | Health          | symptoms, treatment, hospital, wellbeing         |
| language        | Language        | grammar_terms, phonetics, discourse              |
| environment     | Environment     | ecology, climate, recycling                      |
| social          | Social life     | celebrations, going_out, invitations             |
| describing      | Describing      | size_shape, texture, quality, quantity           |

## Rules
- This list can be extended in future, but slugs already in use must not change.
- If a new theme is needed, add it to THEMES.md in a PR before using it in data.
- Sub-themes are optional metadata — a word only needs "theme" to be valid. "sub_theme" is enrichment.
