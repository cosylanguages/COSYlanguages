# COSY Speaking Clubs: Rules & Visual Guidelines

This document serves as the master template for all COSY Speaking Clubs. It ensures consistency across different topics and provides a clear framework for both teachers and students.

## 1. The "COSY Code" (Student Rules)
1.  **Immersion First:** Only the target language is allowed during the session. If you don't know a word, try to describe it or ask the teacher for help in the target language.
2.  **Mistakes are Fuel:** Don't be afraid to make mistakes. The teacher will silently note them down and we will review them together at the end.
3.  **Active Listening:** Respect other participants. Listen as much as you speak.
4.  **Prepare with History:** Before a session, review the "Past Sessions" in the club card to see previous vocabulary and topics.
5.  **Personal Connection:** Try to relate every topic to your own life. Every Round 1 question has a personal "★" variant.

## 2. The 5-Part Session Structure (Teacher's Path)
Every session must follow this flow to maintain a balanced learning experience:

| Part | Title | Focus | Visual Color |
| :--- | :--- | :--- | :--- |
| **1** | **Warm-up** | Introduction, Vocab, Grammar | Peach (`#FAEEE8`) |
| **2** | **Round 1** | Core Topic Questions (Analysis) | Blue (`#E1F5EE`) |
| **3** | **Speak Together** | Visuals, Comparisons, Choices | Purple (`#EEEDFE`) |
| **4** | **Round 2** | Future Focus, Debates, Games | Green (`#EAF3DE`) |
| **5** | **Mistakes** | Linguistic Feedback & Review | Light Red (`#FFF8F5`) |

## 3. Visual Identity & Components
Consistency is key to the user experience. Use these components:

### A. Custom Elements (`vim-`)
- `<vim-choice>`: Use for tabbed content (Vocabulary / Grammar / Questions).
- `<vim-instruction>`: Use for clear session tasks.
- `<vim-blockquote>`: Use for quotes, definitions, or grammar rules.
- `<vim-image>`: Use for visual prompts.

## 4. Vocabulary Format
To ensure compatibility with the dictionary system, all vocabulary must follow this exact format:
`Word – definition. Example: Full sentence using the word.`

## 5. Game Bridges
Sessions should ideally transition into a COSY game to practice the language spontaneously:
- **Philosophy/Quotes** → *Critic's Corner*
- **Opinions/Debates** → *Opinion Arena*
- **General Topics** → *Fluency Flow*
- **Descriptions/Vocab** → *Hot Seat*
